# ANGULAR BLOG

A aplicação desenvolvida trata-se de um blog de cunho pessoal. O desenvolvimento visava o aprendizado de ferramentas como Typescript e Angular, aplicando conceitos de componentização, permitindo assim o gerenciamento de partes independentes do software. 

![imagem da homepage principal do blog feito com Angular](https://github.com/kevin-vogado/angular-blog-/assets/90290277/70211d85-fe6d-4f92-ac5b-b3bb6a78a999)

# Técnicas e Tecnologias Utilizadas

- `Typescript`
- `Javascript`
- `Angular`
- `HTML5`
- `CSS3`
- `Componentização`
